# Energía (env)

Visualizador de proyectos con estética Matrix.

## Características
- Fondo animado con conexiones dinámicas.
- Carga de proyectos desde `pro.json`.
- Interfaz interactiva para añadir nuevos nodos.
- Diseño responsivo.

## Archivos
- `index.html`: Estructura, estilos y lógica principal.
- `pro.json`: Lista de proyectos a mostrar.
